# Steeve Tagasling — Mechanical / Blueprint Portfolio

Industrial blueprint-themed React portfolio with admin dashboard.

## 🚀 Getting Started

```bash
npm install
npm start
```

## 🔧 Build for Production

```bash
npm run build
```

## ⚙️ Admin Panel

Visit your live site → scroll to footer → click **⚙ ADMIN**

Password: `steeve2026`

## 📁 Project Structure

```
src/
├── App.js      ← Main portfolio (Blueprint theme)
├── Admin.js    ← Admin dashboard
├── data.js     ← All portfolio data + localStorage
├── index.js    ← Entry point
└── index.css   ← Base styles
```

## 🌐 Deploy to Vercel / Netlify

Push to GitHub, then connect repo on Vercel or Netlify.
Build command: `npm run build` | Publish dir: `build`
